LGU Dashboard Tutorial

A step-by-step guide to building the Service Request Dashboard in Looker Studio.

Phase 2: Connect to Looker Studio and Create Basic Charts

Step 3Create New Report

  • Go to lookerstudio.google.com
  • Click "Create" → "Report"
  • Click "Google Sheets" under "Connect to data"
  • Find your "LGU_Service_Requests" sheet and click "Select"
  • Click "Add" → "Add to report"

You should now see a blank canvas with your data connected!

Step 4Create First Scorecard

On the right toolbar, click "Add a chart" and select the Scorecard icon (large single number).

  • Metric: Select "Record Count" (shows total service requests)
  • Style:
    • Change background color to light blue
    • Increase font size to 28px
    • Add "Scorecard title" as "Total Service Requests"

Step 5Create More Scorecards

Copy and paste your first scorecard 3 times to create a row.

Phase 3: Add Interactive Filters

Step 8-10Add Controls

Click "Add a control" in the toolbar for each of the following:

  • Dropdown List: Field = "Department" (Width: 200px)
  • Date Range: Default = "Last 30 days" (Width: 250px)
  • Checkbox: Field = "Status" (Orientation: Horizontal)

Phase 4: Create Calculated Fields

Go to Resource → Manage added data sources → Edit LGU_Service_Requests.

Step 11Efficiency Score Formula

CASE
  WHEN `Completion Days` <= 2 THEN "Excellent"
  WHEN `Completion Days` <= 4 THEN "Good" 
  WHEN `Completion Days` <= 6 THEN "Average"
  ELSE "Needs Improvement"
END

Step 12Priority Weight Formula

CASE
  WHEN Priority = "High" THEN 3
  WHEN Priority = "Medium" THEN 2
  WHEN Priority = "Low" THEN 1
  ELSE 0
END

Step 13Weighted Satisfaction Formula

Citizen Satisfaction * Priority Weight
