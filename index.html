Phase 3: Create Calculated Fields

Step 10Create Efficiency Score Field

In the right panel, click "Resource" â†’ "Manage added data sources"

Find "LGU_Service_Requests" and click "EDIT"

Click "ADD A FIELD" in top right

Field name: "Efficiency Score"

Formula: Copy and paste this EXACTLY:

CASE
  WHEN Completion Days <= 2 THEN "Excellent"
  WHEN Completion Days <= 4 THEN "Good" 
  WHEN Completion Days <= 6 THEN "Average"
  ELSE "Needs Improvement"
END

Click "SAVE"

Step 11Create Priority Weight Field

Click "ADD A FIELD" again

Name: "Priority Weight"

CASE
  WHEN Priority = "High" THEN 3
  WHEN Priority = "Medium" THEN 2
  WHEN Priority = "Low" THEN 1
  ELSE 0
END

Click "SAVE"

Step 12Create Weighted Satisfaction Field

"ADD A FIELD"

Name: "Weighted Satisfaction"

Citizen Satisfaction * Priority Weight

Click "SAVE"

Close the data source editor - you'll see these new fields in your field list!

Step 19Create a Calculated Field to Interpret the Range

In Looker Studio â†’ Data Panel, Click Add a field

Name it: BudgetFilter

Paste this formula:

  WHEN BudgetRange = "300000-500000" THEN Budget Allocated >= 300000 AND Budget Allocated < 500000

END
